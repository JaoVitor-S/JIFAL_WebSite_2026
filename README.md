# 🏆 Site JIFAL 2026 (Jogos Internos do IFAL)

Este repositório contém o desenvolvimento do site oficial dos **JIFAL 2026** (Jogos Internos do Instituto Federal de Alagoas – Campus Arapiraca). O projeto foi desenvolvido como parte de uma avaliação na disciplina de Programação Web ministrada pelo professor Italo Carlo.

## 🎯 Objetivo
Criar um site responsivo, limpo e visualmente agradável utilizando HTML5, CSS3 modularizado e Flexbox, com foco na apresentação clara das informações dos Jogos Internos do IFAL. O site demonstra domínio sobre estruturação semântica, estilização, organização de código e responsividade.

## 🗂️ Estrutura do Site
O site contém as seguintes seções:

**1. Página Inicial (Home)**
* Nome do evento: "JIFAL 2026".
* Banner panorâmico de destaque do Campus com overlay escurecido.
* Resumo sobre os Jogos Internos, importância e período dos jogos (13 a 16 de Maio de 2026).
* Destaques do ano atual.
* Menu de navegação responsivo (Hambúrguer).
* Rodapé completo contendo: logo, desenvolvedores, copyright institucional e redes sociais da turma.

**2. Programação**
* Tabela completa e estilizada com dias, horários, categorias, confrontos e locais de prova.
* Divisões inteligentes e responsivas.

**3. Modalidades Esportivas**
* Lista com modalidades (Futsal, Voleibol, Basquetebol, Handebol, Tênis de Mesa e Xadrez).
* Apresentação em cards com ícones do FontAwesome.

**4. Equipes Participantes**
* Exibição das 12 turmas competidoras.
* Cards organizados via Flexbox com bordas estilizadas de acordo com as cores de cada turma.

**5. Galeria de Fotos**
* Registros fotográficos das turmas.
* Efeitos interativos de zoom (hover) estilizados com CSS puro.

**6. Histórico dos Jogos Internos**
* Quadro de medalhas completo e real (Ouro, Prata e Bronze) das edições de 2026, 2025, 2024 e 2023.
* Layout estruturado usando Flexbox em colunas dinâmicas.

**7. Página de Contato**
* Formulário limpo de envio de dúvidas/sugestões.
* Inputs estruturados perfeitamente com Flexbox e select para identificação de vínculo/turma.

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Semântico)
* **CSS3** (Variáveis nativas, animações e modularização)
* **Flexbox** (Alinhamento bidimensional)
* **Google Fonts** (Tipografia 'Outfit')
* **FontAwesome** (Biblioteca de Ícones)

## 💡 Estilo e Design
* **Cores Institucionais:** Uso do verde oficial do IFAL (`#0d6b24` e `#064013`).
* **Componentização:** Padrão de repetição de classes (como rodapé e cabeçalho) para consistência visual.
